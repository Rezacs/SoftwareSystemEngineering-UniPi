Football Social Club

DataOps-based Data Factory for Statistical Classification

Software System Engineering Project - Academic Year 2025/2026

Project Overview

This project implements a modular DataOps pipeline for the Football Social Club platform. The system collects football, medical, and social data, transforms it into prepared sessions, generates learning sets, trains and validates machine learning models, deploys predictions, and evaluates classifier performance.

Flask Python CSV Processing JSON Communication DataOps Machine Learning

System Modules

  • Client-Side System: reads player data and generates JSON messages.
  • Ingestion System: receives data and creates raw sessions.
  • Preparation System: extracts and prepares features.
  • Segregation System: checks balancing and coverage, then generates learning sets.
  • Development System: performs training, validation, and testing.
  • Production System: predicts player score and star rating.
  • Evaluation System: compares expert and classifier labels.

How to Use

Start the Flask application with:

python app.py

Then open:

http://127.0.0.1:5000/

To execute routes such as the full pipeline, use PowerShell:

Invoke-RestMethod -Method Post -Uri http://127.0.0.1:5000/pipeline/run
